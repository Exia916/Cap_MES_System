# UI Standards

The Cap Operations Platform uses a consistent UI design.

## Navigation

Primary navigation groups features by business area.

Examples

Production  
Maintenance  
Quality  
Administration  

## Shared Layout

Typical record pages include:

Main Content Area

Record information

Sidebar

Attachments  
Comments  
Activity History  

## Attachments UI

Attachments are uploaded through the UI and stored in Amazon S3.

The UI provides:

- drag and drop upload
- file preview
- download links
- activity logging

## Global Styling

Shared styling should be placed in:

globals.css

Common reusable classes include:

cards  
buttons  
section headers  
status badges  
metadata rows
